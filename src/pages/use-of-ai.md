# Use of AI

I think it's important to be transparent and intentional about how we use AI. I
am very against AI-generated content (videos, music, articles, etc), and so my
commitment is that **none of the content on this site is or will be AI
generated**. I write everything myself and will continue to do so. (On a related
note, I have been using em dashes long before ChatGPT was invented, so please
don't let that fool you.)

That said, I do use AI in a couple different ways on this site. Mostly I use
GitHub Copoilot for inline auto-complete suggestions for building the layout and
styling. I make extremely minimal use of “agentic” tools; these are generally
only for very small or tedious tasks, or cases where doing something by hand
would be prohibitive. These cases are extremely rare.

If you have any questions, please don't hesitate to [reach out](/contact/).
